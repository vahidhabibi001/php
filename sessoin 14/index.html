while برای اجرای مکرر یک بخش از کد تا زمانی که شرط مشخصی برقرار باشد، استفاده می‌شود. این حلقه تا زمانی که شرط برقرار است، به تکرار ادامه می‌دهد و هنگامی که شرط دیگر برقرار نباشد، از حلقه خارج می‌شود.

The while loop is used to repeatedly execute a section of code as long as a specific condition is true. This loop continues to repeat as long as the condition is met and stops when the condition is no longer true.
